# Skills Overview

Quick reference for all available custom skills.

---

## 1. frontend-design

**Trigger:** Building web components, pages, applications, or any UI

**What it does:**
- Creates production-grade, ultra-premium frontend interfaces
- Applies modern design principles and best practices
- Implements smooth, professional animations
- Builds well-structured, responsive layouts
- Uses premium component patterns (cards, buttons, forms)
- Recommends appropriate libraries (Framer Motion, GSAP, shadcn/ui, Tailwind)

**Key Focus:** Premium feel, clean animations, design systems, accessibility

---

## 2. code-review

**Trigger:** Reviewing PRs, auditing code, improving code quality

**What it does:**
- Performs comprehensive code reviews (security, performance, quality)
- Identifies vulnerabilities and anti-patterns
- Provides constructive, prioritized feedback
- Uses severity levels (Critical, Major, Minor, Nitpick)
- Includes checklists for security, performance, and quality
- Runs complexity analysis and security scans via scripts

**Key Focus:** Security vulnerabilities, performance issues, maintainability, best practices

---

## 3. creative-design

**Trigger:** Artistic projects, experimental designs, generative art, motion graphics

**What it does:**
- Unleashes creative freedom for imaginative designs
- Creates stunning animations and visual effects
- Explores generative art, particles, SVG, Canvas, WebGL
- Experiments with unconventional layouts and typography
- Pushes boundaries while maintaining professional execution
- Uses animation libraries (GSAP, Three.js, p5.js, Lottie)

**Key Focus:** Creative freedom, experimentation, artistic expression, wild imagination with polished delivery

---

## 4. prompt-engineering

**Trigger:** Creating prompts for AI systems, image generation, code tasks

**What it does:**
- Generates ultra-detailed, high-quality prompts
- Applies structured prompt frameworks (CRISP, Chain of Thought)
- Creates prompts for analysis, code generation, image generation
- Uses few-shot learning and role prompting techniques
- Provides templates for different prompt types
- Optimizes prompts for consistency and effectiveness

**Key Focus:** Precision, clarity, structured prompts, best practices for AI interactions

---

## 5. plan-mode

**Trigger:** Planning features, projects, applications, or any multi-step implementation

**What it does:**
- Creates comprehensive implementation plans in .md format
- Breaks work into phases with milestones and tasks
- Estimates time and identifies dependencies
- Assesses risks with mitigation strategies
- Provides templates for projects, features, and migrations
- Includes actionable checklists and success criteria

**Key Focus:** Detailed planning, phased approach, realistic timelines, actionable steps

---
